❤️ Heart Disease Dashboard
📌 Overview

This is a Power BI dashboard for analyzing heart disease data, showing survival rate, death rate, and key health factors.

🚀 Features
Total Alive & Death Analysis
Survival Percentage & Average Age
Age Group-wise Analysis
Diabetes & Health Metrics Insights
Serum & Medical Data Visualization
🛠️ Tools
Power BI
DAX
Data Visualization
📊 Insight
Middle age group has higher cases
Health factors like diabetes impact survival rate
